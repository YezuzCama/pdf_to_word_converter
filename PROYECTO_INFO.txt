📦 PDF to Word Converter
═════════════════════════════════════════

🎯 DESCRIPCIÓN
Una aplicación web para convertir archivos PDF a documentos Word (.docx) editables,
con soporte para dos modos: simple (OCR) y avanzado (con formato).

⚙️ CARACTERÍSTICAS PRINCIPALES
═════════════════════════════════════════

✨ Conversión en DOS MODOS:

  1. 📄 MODO SIMPLE
     - Extracción de texto mediante OCR (Tesseract)
     - Rápido y eficiente
     - Ideal para documentos con mucho texto
     - Formato básico pero limpio

  2. 🎨 MODO AVANZADO
     - Intenta preservar formato original
     - Incluye miniaturas de las páginas originales
     - Detección automática de encabezados
     - Mejor para documentos con diseño especial

✨ INTERFAZ MODERNA:
   - Drag & drop para cargar archivos
   - Indicadores de progreso en tiempo real
   - Descarga automática del archivo Word
   - Responsive design (móvil + desktop)
   - Validación de archivos en el cliente

✨ TECNOLOGÍA:
   - Backend: FastAPI (Python asincrónico)
   - Frontend: HTML5 + CSS3 + JavaScript vanilla
   - OCR: Tesseract
   - PDF: pdf2image + Poppler
   - Word: python-docx

🔌 USO DE LA API
═════════════════════════════════════════

ENDPOINT: POST /api/convert

Parámetros:
  - file: Archivo PDF (form-data)
  - mode: 'simple' o 'advanced' (form-data)

Ejemplo con cURL:
  curl -F "file=@documento.pdf" \
       -F "mode=simple" \
       http://localhost:8000/api/convert \
       -o documento.docx

Ejemplo con Python:
  import requests
  
  with open('documento.pdf', 'rb') as f:
      files = {'file': f}
      data = {'mode': 'simple'}
      response = requests.post(
          'http://localhost:8000/api/convert',
          files=files,
          data=data
      )
  
  with open('documento.docx', 'wb') as f:
      f.write(response.content)


🐛 SOLUCIÓN DE PROBLEMAS
═════════════════════════════════════════

❌ Error: "TesseractNotFoundError"
   → Instalar Tesseract OCR desde:
     https://github.com/UB-Mannheim/tesseract/wiki

❌ Error: "No module named 'pdf2image'"
   → Ejecutar: pip install --upgrade pdf2image

❌ Error: "ModuleNotFoundError: No module named 'fastapi'"
   → Asegúrate de activar el entorno virtual
   → Ejecutar: pip install -r requirements.txt

❌ El servidor no inicia
   → Verificar que puerto 8000 no esté en uso
   → Ver log del servidor para más detalles

❌ El OCR no funciona bien
   → Usar PDFs con buena calidad
   → OCR con idiomas diferentes: editar config.py
