🎯 FUNCIONALIDAD PRINCIPAL
────────────────────────────────────────────────────────────────────────────

  ✓ Interfaz web moderna
  ✓ Drag & drop de archivos
  ✓ Descarga automática
  ✓ Validación de archivos

⚙️ TECNOLOGÍA UTILIZADA
────────────────────────────────────────────────────────────────────────────

Backend:
  • FastAPI 0.109         → Framework web moderno y rápido
  • Uvicorn              → Servidor ASGI de producción
  • Python 3.8+          → Lenguaje de programación

OCR y Procesamiento:
  • Tesseract            → Motor OCR (instalación separada)
  • pdf2image            → Conversión PDF a imagen
  • pytesseract          → Interfaz Python para Tesseract
  • Pillow               → Procesamiento de imágenes

Documentos Word:
  • python-docx          → Crear/editar documentos Word

Frontend:
  • HTML5                → Estructura
  • CSS3                 → Estilos modernos
  • JavaScript Vanilla   → Interactividad (sin dependencias)


🚀 INSTALACIÓN (3 PASOS)
────────────────────────────────────────────────────────────────────────────

PASO 1: Instalar Tesseract OCR
  • Descargar desde: https://github.com/UB-Mannheim/tesseract/wiki
  • Instalar en: C:\Program Files\Tesseract-OCR (Windows)

PASO 2: Ejecutar instalador
  • Windows: Doble clic en install.bat
  • Otros: python -m venv venv && pip install -r requirements.txt

PASO 3: Ejecutar aplicación
  • Windows: Doble clic en run.bat
  • Otros: python main.py
  • Abrir: http://localhost:8000


✨ CARACTERÍSTICAS PRINCIPALES
────────────────────────────────────────────────────────────────────────────

1. DOS MODOS DE CONVERSIÓN
   ├─ Simple: Extracción de texto puro (OCR)
   │  ├─ Rápido (10-20 segundos)
   │  ├─ Formato limpio
   │  └─ Ideal para documentos de texto
   │
   └─ Avanzado: Preservación de formato
      ├─ Más lento (30-60 segundos)
      ├─ Incluye imágenes de páginas
      ├─ Detección de encabezados
      └─ Ideal para documentos con diseño

2. INTERFAZ MODERNA
   ├─ Drag & drop intuitivo
   ├─ Indicadores de progreso
   ├─ Diseño responsive (móvil/desktop)
   ├─ Validación en cliente
   └─ Descarga automática

3. API REST
   ├─ Endpoint POST /api/convert
   ├─ Parámetros: file (PDF) + mode (simple/advanced)
   ├─ Respuesta: Archivo Word (.docx)
   └─ Usable desde cualquier cliente HTTP

4. AUTOMATIZACIÓN
   ├─ Scripts de instalación
   ├─ Verificación del sistema
   ├─ Ejemplos de uso
   └─ Procesamiento por lotes

🎓 EJEMPLOS DE USO
────────────────────────────────────────────────────────────────────────────

Desde Interfaz Web:
  1. Abrir http://localhost:8000
  2. Seleccionar modo (Simple/Avanzado)
  3. Cargar PDF (drag & drop o click)
  4. Click en "Convertir a Word"
  5. Descargar automáticamente

Desde API con cURL:
  curl -F "file=@documento.pdf" \
       -F "mode=simple" \
       http://localhost:8000/api/convert \
       -o documento.docx

Desde Python:
  Ver archivo: examples.py
  Incluye 4 ejemplos completos

Desde Línea de Comandos:
  python examples.py
  Menú interactivo para probar
