🚀 CÓMO USAR (3 PASOS)
════════════════════════════════════════════════════════════════════════════

PASO 1 - INSTALAR TESSERACT OCR (OBLIGATORIO)
──────────────────────────────────────────────
1. Ir a: https://github.com/UB-Mannheim/tesseract/wiki
2. Descargar: tesseract-ocr-w64-setup-v5.x.x.exe
3. Ejecutar instalador: Siguiente → Siguiente → Finalizar
4. Verifica: Abrir cmd y escribe: tesseract --version

PASO 2 - INSTALAR DEPENDENCIAS
──────────────────────────────
Opción A (Windows - Automática):
  1. Doble clic en: install.bat
  2. Esperar a completar (5-10 minutos)
  3. Listo

Opción B (Manual):
  1. python -m venv venv
  2. venv\Scripts\activate
  3. pip install -r requirements.txt

PASO 3 - EJECUTAR APLICACIÓN
──────────────────────────────
Opción A (Windows - Automática):
  1. Doble clic en: run.bat
  2. Abrir: http://localhost:8000

Opción B (Manual):
  1. venv\Scripts\activate
  2. python main.py
  3. Abrir: http://localhost:8000
